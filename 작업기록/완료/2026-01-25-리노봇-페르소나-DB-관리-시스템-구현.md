## 📋 작업 개요

### 문제점
- Rino 봇 페르소나 파일(YAML)이 Docker 이미지 내부에 포함되어 있어 수정 시 매번 재빌드 및 재배포 필요
- 페르소나 변경 이력 추적 불가
- 버전 관리 없음

### 해결 방안
- PostgreSQL DB에 페르소나 저장
- 웹 UI로 실시간 편집
- 버전 관리 및 변경 이력 추적
- 하위 호환성 유지 (YAML fallback)

### 1. 백엔드 구현 (8개 작업)

#### 1.1 DB 스키마 생성
**파일**: `server/database/migrations/add-ai-personas-tables.sql`

**테이블**:
- `ai_personas`: 페르소나 정의 및 버전 관리
  - 컬럼: id, agent_id, variant, version, name, description
  - 성격: personality_traits[], personality_tone, personality_formality, personality_emoji_usage
  - 대화: voice_language, voice_style, voice_examples[]
  - 행동: behavior_do[], behavior_dont[], behavior_response_guidelines[]
  - 지식: knowledge_areas[], knowledge_limitations[]
  - 시스템 프롬프트: system_prompt_template (TEXT)
  - 상태: is_active, is_archived
  - 메타: created_by, updated_by, created_at, updated_at, activated_at
  - 제약: UNIQUE(agent_id, variant, version)

- `ai_persona_change_logs`: 변경 이력 감사 로그
  - 컬럼: id, persona_id, action, previous_version, performed_by, notes, metadata, created_at

**인덱스**:
- `idx_personas_active`: (agent_id, variant, is_active) WHERE is_active = true

## 🎯 아키텍처 설계

### DB 스키마 설계
```sql
ai_personas (페르소나 정의)
├── id (SERIAL PRIMARY KEY)
├── agent_id + variant + version (UNIQUE)
├── is_active (활성 버전, 1개만 true)
├── is_archived (소프트 삭제)
└── 외래키: created_by, updated_by → users(id)

ai_persona_change_logs (변경 이력)
├── id (SERIAL PRIMARY KEY)
├── persona_id → ai_personas(id)
├── action (created, activated, archived)
├── previous_version (활성화 시 이전 버전)
└── 외래키: performed_by → users(id)
```

### 백엔드 아키텍처
```
Client Request
    ↓
Express Router (aiAdminRoutes.ts)
    ↓
PersonaService (비즈니스 로직)
    ↓
PostgreSQL (ai_personas, ai_persona_change_logs)
    ↓
PersonaLoader (캐시 + DB + YAML fallback)
    ↓
DbPersonaLoader (DB 조회)
    ↓
AIPersona (타입)
```

### 프론트엔드 아키텍처
```
/admin/personas (목록)
    ↓
PersonaList Component
    ↓ 클릭
/admin/personas/[agentId]/[variant] (편집)
    ↓
PersonaEditClient (Client Component)
    ├─ PersonaEditor (편집 폼)
    │   └─ 500ms debounce → API 호출
    └─ PersonaPreview (미리보기)
        └─ 실시간 업데이트
```
