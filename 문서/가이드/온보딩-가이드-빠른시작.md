## 2. GitHub SSH 키 설정

```bash
# SSH 키 생성
ssh-keygen -t ed25519 -C "your_email@example.com"
# Enter 3번 (기본 경로, 비밀번호 없음)

# SSH 에이전트 시작
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

# 공개 키 출력 → GitHub에 등록
cat ~/.ssh/id_ed25519.pub
```

**GitHub에 등록**: Settings → SSH and GPG keys → New SSH key

```bash
# 연결 테스트
ssh -T git@github.com
# 출력: Hi username! You've successfully authenticated...
```

## 4. 환경변수 설정 (Doppler)

### 4.1 Doppler CLI 설치

```bash
# Doppler CLI 설치
sudo apt-get update && sudo apt-get install -y apt-transport-https ca-certificates curl gnupg
curl -sLf --retry 3 --tlsv1.2 --proto "=https" 'https://packages.doppler.com/public/cli/gpg.DE2A7741A397C129.key' | sudo gpg --dearmor -o /usr/share/keyrings/doppler-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/doppler-archive-keyring.gpg] https://packages.doppler.com/public/cli/deb/debian any-version main" | sudo tee /etc/apt/sources.list.d/doppler-cli.list
sudo apt-get update && sudo apt-get install doppler

# 설치 확인
doppler --version
```

### 4.2 환경변수 다운로드

팀 리더에게 각 프로젝트의 Doppler 서비스 토큰을 요청한 후 실행:

```bash
# WBHubManager
cd ~/GitHub/WBHubManager
DOPPLER_TOKEN=<팀리더에게_받은_토큰> doppler secrets download --no-file --format env > .env.local

# WBSalesHub
cd ~/GitHub/WBSalesHub
DOPPLER_TOKEN=<팀리더에게_받은_토큰> doppler secrets download --no-file --format env > .env.local

# WBFinHub
cd ~/GitHub/WBFinHub
DOPPLER_TOKEN=<팀리더에게_받은_토큰> doppler secrets download --no-file --format env > .env.local
```

**참고**: 토큰은 `~/GitHub/WHCommon/env.doppler` 파일에 보관됩니다.

## 5. 데이터베이스 설정 (Docker PostgreSQL)

```bash
# PostgreSQL 컨테이너 실행
docker run -d \
  --name workhub-postgres \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=postgres \
  -p 5432:5432 \
  postgres:15

# 데이터베이스 생성
docker exec -it workhub-postgres psql -U postgres -c "CREATE DATABASE hubmanager;"
docker exec -it workhub-postgres psql -U postgres -c "CREATE DATABASE saleshub;"
```
