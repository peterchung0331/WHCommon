## Executive Summary

한국 토큰증권(STO) 시장은 자본시장법·전자증권법 개정안이 국회 정무위를 통과하며 **2027년 1월 시행**을 앞두고 있다. 국내 STO 시장 규모는 2026년 119조원에서 2030년 367조원으로 성장이 전망된다. 글로벌 RWA(실물자산 토큰화) 시장은 2025년 $186B 온체인 규모를 기록하며 전년 대비 3배 성장했고, 2031년까지 $18.74T 규모로 확대될 것으로 예측된다.

**핵심 수치**:

| 지표 | 값 |
|------|-----|
| **국내 STO 시장 (2026)** | 119조원 |
| **국내 STO 시장 (2030)** | 367조원 |
| **글로벌 RWA 온체인 (2025)** | $186B |
| **글로벌 토큰화 시장 (2031)** | $18.74T |
| **법 시행 예정일** | 2027. 01. |
| **주요 플레이어** | 신한투자, 한투, KB, 미래에셋, 키움, 뮤직카우, 카사 |
| **시장 진입 난이도** | 중 (VASP 보유 시 유리) |

**웨이브릿지 핵심 기회**:
웨이브릿지는 VASP 라이선스, Fireblocks 커스터디 파트너십, 솔라나 재단 원화 스테이블코인 MOU, 그리고 Wavebridge Prime 인프라를 보유하고 있어, STO 시장의 **수탁·보관(Custody)**, **유동성 공급(Liquidity Provider)**, **토큰화 엔진 제공** 영역에서 차별화된 포지션을 확보할 수 있다.

## 1. 개념 정의 및 시장 구분

### 1.1 STO(Security Token Offering, 토큰증권)

토큰증권은 **분산원장기술(DLT)을 활용하여 발행·유통되는 증권**이다. 기존 전자증권법이 중앙집중식 계좌 시스템만 허용하던 것을 확장하여, 블록체인 기반 분산원장을 법적 효력이 있는 증권 기록 방식으로 인정한다.

- **규제**: 자본시장법 적용, 금융당국 직접 규제
- **블록체인**: 주로 프라이빗/허가형 블록체인
- **대상 자산**: 증권성 있는 자산에 한정 (부동산, 미술품, IP, 채권 등)
- **핵심 원칙**: 발행과 유통의 엄격한 분리

### 1.2 RWA(Real World Assets, 실물자산 토큰화)

RWA는 **부동산, 채권, 미술품, 원자재 등 실물 자산을 블록체인 토큰으로 디지털화**하는 것이다. 증권형 자산뿐 아니라 모든 종류의 실물 자산을 온체인에서 거래 가능하게 하는 포괄적 개념이다.

- **규제**: 자산 유형별 상이, 규제 불확실성 존재
- **블록체인**: 주로 퍼블릭 블록체인 (이더리움, 솔라나 등)
- **대상 자산**: 증권 포함 모든 실물 자산
- **특징**: DeFi 연계, 탈중앙성이 강함

### 1.3 STO vs RWA 비교

| 구분 | STO (토큰증권) | RWA (실물자산 토큰화) |
|------|---------------|---------------------|
| **개념 범위** | 하위 개념 (증권 한정) | 상위 개념 (모든 실물 자산) |
| **규제** | 명확 (자본시장법) | 불확실 (자산별 상이) |
| **블록체인** | 프라이빗/허가형 | 퍼블릭 (이더리움, 솔라나) |
| **중앙화** | 제도화·중앙화 | 탈중앙화 지향 |
| **유통** | 인가 장외거래소 | DEX + CEX 혼합 |
| **투자자** | 국내 규제 내 투자자 | 글로벌 투자자 |
| **대표 사례** | 부동산 조각투자, IP 토큰 | BlackRock BUIDL, MakerDAO |

### 1.4 시장 세분화

| 세그먼트 | 설명 | 주요 자산 |
|---------|------|----------|
| **토큰화 증권** | 법적 증권을 블록체인으로 발행 | 부동산, 채권, 주식, IP |
| **토큰화 예금/MMF** | 은행예금·단기금융상품 토큰화 | MMF, 정기예금 |
| **토큰화 상품** | 실물 상품 토큰화 | 금, 원자재, 미술품 |
| **스테이블코인** | 법정화폐 기반 토큰 | USDT, USDC, 원화 스테이블코인 |
| **토큰화 국채** | 국채를 온체인 토큰으로 | US Treasury (BUIDL) |

## 5. 기술 동향

### 5.1 블록체인 인프라

| 유형 | 특징 | 한국 STO 적용 | RWA 적용 |
|------|------|-------------|---------|
| **프라이빗/허가형** | 허가된 참여자만 접근, 높은 규제 적합성 | 주력 (증권사 자체 메인넷) | 제한적 |
| **퍼블릭** | 개방형, 글로벌 유동성 접근 | 제한적 | 주력 (이더리움, 솔라나) |
| **하이브리드** | 프라이빗 발행 + 퍼블릭 유통 | 향후 유력 | 성장 중 |

- **미래에셋**: 하나금융·SKT와 자체 STO 메인넷 개발
- **신한투자**: 솔라나(퍼블릭) 기반 접근
- **한국예탁결제원**: 토큰증권 총량관리 테스트베드 구축

### 5.2 커스터디/수탁 기술

STO/RWA 시장에서 **기관급 커스터디**는 필수 인프라:

| 솔루션 | 특징 | 한국 파트너 |
|--------|------|-----------|
| **Fireblocks** | 글로벌 1위 기관 커스터디, MPC 기술 | 웨이브릿지, 신한투자증권 |
| **코스콤** | 한국 증권 IT 인프라 | DB증권 시범사업 |
| **한국예탁결제원** | 토큰증권 총량관리 | 전 증권사 |

### 5.3 스테이블코인 연계

원화 스테이블코인은 STO/RWA의 **결제 레이어**로 핵심:

- **웨이브릿지 + 솔라나 재단 MOU** (2025.10):
  - 원화 스테이블코인 발행·운영·검증 토큰화 엔진 공동 개발
  - 화이트리스트 관리, 전송 통제, 검증 절차 내장
  - MMF 토큰화 프로젝트 공동 추진

- 2026년 제도권 진입이 주목되는 영역 — 원화 스테이블코인 활용 논의 활발화

### 5.4 토큰화 엔진

토큰화 엔진은 실물 자산을 블록체인 토큰으로 변환하는 핵심 기술:

| 기능 | 설명 |
|------|------|
| **발행(Issuance)** | 자산을 토큰으로 변환, 스마트 컨트랙트 배포 |
| **검증(Verification)** | KYC/AML, 적격 투자자 확인 |
| **화이트리스트** | 허가된 주소만 거래 가능 |
| **전송 통제** | 규제 요건에 따른 전송 제한 |
| **리포팅** | 세무·회계·규제 보고 자동화 |

웨이브릿지의 솔라나 기반 토큰화 엔진은 이 모든 기능을 포함할 예정.

## 7. 웨이브릿지 사업 기회 분석

### 7.1 기존 역량 맵핑

| **Wavebridge Prime** | 법인·기관 전용 프라임 브로커리지 | 기관 투자자 접점 | 토큰화 자산 중개 |
| **Fireblocks 파트너십** | 기관급 커스터디·수탁 | 토큰증권 수탁 | 토큰화 자산 보관 |
| **솔라나 MOU** | 원화 스테이블코인 엔진 + MMF 토큰화 | 결제 인프라 | 퍼블릭 체인 토큰화 |

### 7.2 STO 시장 내 포지셔닝

STO 밸류체인에서 웨이브릿지가 진입할 수 있는 영역:

```
[발행인] → [발행 플랫폼] → [유통(장외거래소)] → [수탁/보관] → [투자자]
           (증권사)        (KDX/NXT)           ★ 여기     ★ 여기
```

| 역할 | 가능성 | 근거 |
|------|--------|------|
| **수탁/커스터디** | 높음 | Fireblocks 파트너십, VASP 라이선스 |
| **유동성 공급(LP)** | 높음 | Wavebridge Prime 인프라, 신한투자 협력 경험 |
| **결제 인프라** | 높음 | 원화 스테이블코인 엔진 (솔라나 MOU) |
| **가치 평가** | 중 | Wave Pricing Engine, Analytic Solver |
| **직접 발행** | 낮음 | 증권사 인가 필요 (VASP만으로 불충분) |
| **장외거래소 운영** | 낮음 | 별도 인가 필요, 이미 3파전 진행 중 |

### 7.3 RWA 토큰화 기회

| 기회 영역 | 상세 | 우선순위 |
|----------|------|---------|
| **원화 스테이블코인** | 솔라나 기반 발행·운영·검증 엔진 (MOU 체결 완료) | 1순위 |
| **MMF 토큰화** | 단기금융상품 온체인 운용 (솔라나 MOU 포함) | 1순위 |
| **토큰화 채권** | 기관 채권의 블록체인 발행·유통 중개 | 2순위 |
| **크로스보더 RWA 유통** | EU 법인(MiCA) 활용한 한국-EU 간 토큰화 자산 유통 | 2순위 |
| **토큰화 엔진 B2B** | 토큰화 엔진을 증권사/금융기관에 화이트라벨 제공 | 3순위 |

### 7.4 진입 전략 및 차별화

**차별화 포인트**: 한국에서 VASP + 글로벌 라이선스(EU) + 기관급 커스터디(Fireblocks) + 퍼블릭체인 토큰화 엔진(솔라나)을 모두 보유한 사업자는 웨이브릿지가 유일하다.

**전략 로드맵**:

| 단계 | 시기 | 액션 |
|------|------|------|
| **Phase 1: 기반 구축** | 2026 H1 | 원화 스테이블코인 엔진 완성, MMF 토큰화 PoC |
| **Phase 2: STO 인프라** | 2026 H2 | 증권사 파트너십 확대 (수탁/LP 역할), 토큰증권 커스터디 서비스 론칭 |
| **Phase 3: 법 시행 대응** | 2027 H1 | STO 시행과 동시에 수탁·유동성 서비스 본격화 |
| **Phase 4: 크로스보더** | 2027 H2+ | EU MiCA 활용, 한국-EU 간 토큰화 자산 크로스보더 유통 |

## 10. 결론 및 권고사항

한국 STO 시장은 2027년 법 시행을 앞두고 본격적인 성장 전야에 있으며, 글로벌 RWA 시장은 이미 폭발적 성장을 시작했다. 웨이브릿지는 VASP + 글로벌 라이선스 + Fireblocks + 솔라나 토큰화 엔진이라는 독보적 조합으로, 제도권(STO)과 글로벌(RWA)을 연결하는 브릿지 역할을 수행할 최적의 위치에 있다.

**핵심 3가지 권고**:

1. **원화 스테이블코인 + MMF 토큰화 가속화**: 솔라나 MOU를 바탕으로 2026년 상반기 내 토큰화 엔진 MVP 완성. 이는 STO 결제 인프라와 RWA 유동성의 기반이 된다.

2. **STO 수탁·유동성 서비스 선점**: Fireblocks 기반 기관급 토큰증권 커스터디 서비스를 2026년 하반기 론칭하고, 증권사(신한투자, 한투, KB)와 수탁·LP 계약을 선제 체결한다. 2027년 법 시행 시점에 즉시 서비스가 가능해야 한다.

3. **크로스보더 RWA 브릿지 구축**: EU MiCA CASP 라이선스를 활용하여 한국-EU 간 토큰화 자산 유통 통로를 구축한다. 이는 국내 증권사가 단독으로 제공하기 어려운 글로벌 역량으로, 웨이브릿지만의 차별화 요소가 된다.
