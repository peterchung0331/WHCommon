# Admin — 컨텍스트

## 개요

회사 운영 기반 문서: 양식, 체결 계약서, 회사소개서 원본, 회사정보 문서. 규제 제출 및 내부 운영의 기초 자료.

## 파일 구조 (109 files)

```
Admin/
├── 문서양식/          — 내부 업무 양식 (보고서, 제안서 등 템플릿)
├── 체결계약서/        — 체결 완료 계약서 보관
├── 회사소개서/        — 회사소개서 마스터 파일
└── 회사정보문서/      — 사업자등록증, 등기부등본, 정관, VASP 인증 등
```
