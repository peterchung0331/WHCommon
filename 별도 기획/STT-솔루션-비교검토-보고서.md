# 음성-텍스트 변환(STT) 솔루션 비교검토 보고서

> **작성일**: 2026. 02. 19.
> **작성자**: 정피터
> **목적**: SalesHub 영업 녹음 전사 기능 도입을 위한 STT 솔루션 비교 및 최종 선정

---

## 1. 도입 배경

영업팀의 고객 미팅/통화 녹음을 텍스트로 변환하고, AI(Claude)로 요약하는 파이프라인을 SalesHub에 구축하고자 합니다.

**파이프라인**: 모바일 녹음 → STT 전사 → Claude API 요약 → SalesHub 저장

핵심 요구사항:
- **한국어 인식 정확도**가 최우선 (영업 전문용어, 고객명, 금액 등)
- SalesHub 백엔드(Node.js)와 API 연동 가능
- 화자분리(고객/담당자 구분) 지원

---

## 2. 후보 솔루션 비교

한국어 음성인식 벤치마크(AI-Hub 테스트셋, CER 기준)를 포함한 6개 솔루션을 비교했습니다.

> **CER**(Character Error Rate): 문자 단위 오류율. 낮을수록 정확함.
> 출처: [Awesome-Korean-Speech-Recognition](https://github.com/rtzr/Awesome-Korean-Speech-Recognition)

### 2.1 성능 및 가격 비교

| 솔루션 | 한국어 CER | 시간당 비용 | 화자분리 | API 연동 |
|--------|-----------|-----------|---------|---------|
| **리턴제로 RTZR STT** | **5.91%** | **1,000원** | O | REST API |
| 네이버 CLOVA Speech | 7.52% | ~1,800원 | O | REST API |
| Azure Batch STT | 10.88% | ~470원 | O | REST + SDK |
| OpenAI Whisper | 11.39% | ~468원 | O | REST API |
| Gemini 2.0 Flash | 16.58% | ~130원 | X | 토큰 기반 |
| Fireflies.ai | 미공개 | 260~780원 | O | 구독형 |

> Fireflies.ai: 자체 STT 엔진 없이 외부 엔진(Whisper/Google) 조합 사용. 미팅 노트 플랫폼 특화.
> Gemini: 범용 LLM의 오디오 입력 기능으로, 전용 STT 대비 정확도 열위.

### 2.2 가격-정확도 포지셔닝

| 구간 | 솔루션 | 평가 |
|------|--------|------|
| **고정확 + 합리적 가격** | **RTZR (5.91%, 1,000원/h)** | 최적 균형점 |
| 고정확 + 고가격 | CLOVA (7.52%, 1,800원/h) | 정확도 낮으면서 80% 비쌈 |
| 저정확 + 저가격 | Gemini (16.58%, 130원/h) | 영업 녹음에 부적합 |
| 중정확 + 저가격 | Azure/Whisper (~11%, ~470원/h) | 한국어 특화 부족 |

---

## 3. 정확도 차이의 실질적 영향

1분 분량(약 300자) 영업 녹음 전사 시:

| | RTZR (CER 5.91%) | Gemini (CER 16.58%) |
|--|-------------------|---------------------|
| 오류 글자 수 | ~18자 | ~50자 |
| 틀린 단어 수 | ~5개 | ~15개 |
| 체감 | 간간이 오타, 바로 요약 가능 | 문장마다 1~2개 오류, 수동 교정 필요 |

**영업 녹음 특성상 고객명, 금액, 수수료율, 회사명 등 핵심 정보의 오류는 후속 AI 요약 품질에 직접적 영향**을 미칩니다. 저가 솔루션의 오류를 사람이 교정하는 인건비를 고려하면 정확도 우선 선택이 총비용 기준으로도 유리합니다.

## 5. 최종 선정: 리턴제로 RTZR STT

### 5.1 선정 사유

| 기준 | RTZR 평가 |
|------|----------|
| **한국어 정확도** | CER 5.91% — 전체 1위 (2위 대비 1.6%p 우위) |
| **가격 경쟁력** | 1,000원/시간 — CLOVA 대비 44% 저렴 |
| **API 연동성** | REST API + Node.js 공식 튜토리얼 제공 |
| **화자분리** | 내장 지원 (자동 화자 수 예측) |
| **부가 기능** | 간투어 필터, 비속어 필터, 키워드 부스팅, 개인정보 필터 |
| **무료 크레딧** | 가입 시 600분(10시간) 제공 — 초기 검증 가능 |

### 5.2 결론

리턴제로 RTZR STT는 **한국어 정확도 1위이면서 가격도 가장 합리적인 솔루션**입니다. 정확도가 높을수록 후속 AI 요약 품질이 올라가고, 사용자의 수동 교정 부담이 줄어들어 **총 운영비용(TCO) 관점에서도 최적**입니다.

연간 약 80만 원 수준의 비용으로 영업팀의 녹음 전사 + AI 요약 파이프라인을 구축할 수 있으며, 이는 영업 미팅 기록 누락 방지 및 고객 인사이트 축적에 기여할 것으로 판단됩니다.
