## 2. 사업 영역

웨이브릿지는 한국 **VASP(가상자산사업자) 라이선스** 하에서 법인·기관 대상 디지털자산 서비스를 제공한다.

### 핵심 사업

| 사업 | 설명 |
|------|------|
| **프라임 브로커리지** | 기관·법인 전용 디지털자산 매매·중개·수탁·보관·블록딜 |
| **온오프램프** | 법인 디지털자산 매매 중개 (법정화폐 ↔ 디지털자산) |
| **커스터디 연계** | Fireblocks 기반 수탁 기술 제공 |
| **사무관리** | 세무·회계 지원, 월별/분기별 리포팅 |
| **리스크 관리** | 파생상품 공정가격 평가, 포트폴리오 리스크 분석 |

### 신규/추진 중 사업

| 사업 | 상태 | 설명 |
|------|------|------|
| **원화 스테이블코인** | MOU 체결 (2025.10) | 솔라나 재단과 발행·운영·검증 토큰화 엔진 공동 개발 |
| **MMF 토큰화** | 검토 중 | 머니마켓펀드 블록체인 운용 실험 |
| **BTC 현물 ETF** | 공동 연구 | 신한투자증권·Fireblocks와 ETF 인프라 구축 |
| **구조화 금융상품** | 준비 중 | 홍콩 현지 금융기관과 디지털자산 구조화 상품 |

## 3. 제품/서비스

### Wavebridge Prime (구 Dolfin)

법인·기관 전용 프라임 브로커리지 플랫폼.

| 특징 | 설명 |
|------|------|
| **최적 실행(Best Execution)** | 거래소 오더북 직접 주문이 아닌 최적화된 거래 실행 |
| **통합 인프라** | 글로벌 거래소, 커스터디, 월렛, 온체인 검증 모듈을 단일 채널로 통합 |
| **사무관리** | 기관 전용 세무·회계 리포팅 |
| **대차/OTC** | 대차 및 장외거래 기능 |
| **글로벌 네트워크** | 글로벌 기관 네트워크 연결 |

- 2025.03: Dolfin 플랫폼 출시
- 2025.12: Dolfin → Wavebridge Prime으로 리브랜딩

### Alfred Pro

파생상품 평가 및 투자 전략 실행이 가능한 거래 플랫폼.

### Wave Pricing Engine

파생상품 공정 가격 평가 및 리스크 관리 엔진.

### Analytic Solver

바젤3 기준 부합 자산평가·위험관리 솔루션. 신한은행, 하나은행 등과 공급 계약 체결.

## 10. WorkHub 시스템과의 관계

웨이브릿지의 내부 업무 시스템이 **WorkHub**이며, 다음 허브로 구성:

| 허브 | 역할 |
|------|------|
| **WBHubManager** | 메인 관리 (SSO, 사용자/허브 관리) |
| **WBSalesHub** | 영업 CRM + Reno AI 봇 |
| **WBFinHub** | 재무/거래 관리 (Fireblocks 연동) |

> 상세 기술 아키텍처는 `아키텍처/overview.md` 참조
